How to Use

🚀 Welcome to CodeMastery

Practice. Test. Master. Repeat.

💻 Practice Mode

Type code and learn step-by-step. The blurred solution reveals letter-by-letter as you type.

  • 🟢 Correct characters turn green
  • ⚪ Wrong characters stay white
  • 👀 Helps you learn syntax precisely

📋 Test Mode

Solve problems independently without hints.

  • 🎯 No step-by-step guidance
  • ✅ Output shown only when correct
  • 🏁 Simulates real coding tests

🔁 Progress System

  • 📌 Your last problem is saved automatically
  • 🔙 Returning continues from where you left
  • 📚 Works in both Practice & Test modes

🧩 Problem Selection

Choose problems manually or continue sequentially in both modes.

✨ Final Note

Stay consistent. Focus on accuracy.

Happy Learning 🚀
